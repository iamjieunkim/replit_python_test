📌 파이썬의 숫자형 

파이썬에서 숫자 자료형으로는 대표적으로 정수형과 실수형이 있다.


✔ 정수형 : int / 소수점이 없고 사람의 수를 세거나 카드 개수를 세는 등의 계산에 사용

✔ 실수형 : float / 소수점이 포함된 실수. 분수나 정확한 측정값에 이용되고 평균을 구할 때 많이 쓰임

📌Boolean

참(True)과 거짓(False)을 나타내는 자료형이에요.
